Préparer une demande utile

Les sollicitations les plus pertinentes relient un sujet, un contexte, un format attendu, un objectif et un délai éventuel. Cette préparation permet d'identifier rapidement le bon cadre d'échange.

  • Identité / organisme : institution, structure, média, réseau ou cadre de l'échange.
  • Objet : échange, contribution, conférence, atelier, formation, projet ou demande institutionnelle.
  • Contexte : public concerné, problématique, territoire ou cadre de discussion.
  • Lien avec les axes OBY : mer, littoral, médiation, gouvernance, innovation ou prospective.
  • Échéance éventuelle : période, calendrier ou niveau d'urgence si pertinent.
  • Format attendu : entretien, réunion, contribution écrite, intervention, atelier ou échange exploratoire.
  • Coordonnées de retour : contact, fonction et moyen de réponse approprié.

Travaux, terrains et enseignement

Les demandes portant sur un travail, un terrain, une visite, une intervention, une vacation ou un support pédagogique doivent préciser le cadre exact, le format attendu et le niveau de diffusion souhaité.

Pour proposer une intervention, un séminaire, un atelier, une formation courte, une collaboration académique ou une contribution à un projet, merci de préciser le contexte, l’objectif, le public, la durée, le calendrier et le cadre envisagé.

Travail ou support

Indiquer le titre concerné, le contexte d'usage, le public destinataire et le type de document attendu.

Terrain ou visite

Préciser le lieu, le cadre, la période, les objectifs d'observation et les éventuelles contraintes de diffusion.

Intervention ou module

Présenter le thème, le format, la durée, le public, les objectifs pédagogiques et le lien avec les axes OBY.
